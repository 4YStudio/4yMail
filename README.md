### 🌟 Introduction / 简介

**4yMail** is a desktop email client built with a focus on user experience and aesthetics. It combines the power of Electron with a stunning "Liquid Glass" (Glassmorphism) design system and fluid animations.

**4yMail** 是一款专注于用户体验与视觉美学的桌面邮件客户端。它将 Electron 的强大功能与令人惊叹的“液态玻璃”（Glassmorphism）设计系统以及灵动的交互动画深度融合。

---

### ✨ Key Features / 核心特性

- **💎 Liquid Glass UI / 液态玻璃 UI**  
  A premium, translucent interface with dynamic gradients and glassmorphism effects.  
  极具质感的半透明界面，配合动态渐变背景与玻璃拟态效果，提供极致的视觉享受。

- **🎬 Fluid Animations / 灵动动效**  
  Staggered list loading, smooth route transitions, and interactive micro-animations.  
  支持阶梯式列表加载、顺滑的路由转场以及细腻的交互微动效。

- **🚀 Multi-account Support / 多账号支持**  
  Manage multiple IMAP/SMTP accounts seamlessly with quick-switch capabilities.  
  无缝管理多个 IMAP/SMTP 账号，支持快速切换与统一视图管理。

- **🌗 Smart Themes / 智能主题**  
  Seamlessly switch between sophisticated Dark and Light modes.  
  深度优化的深色与浅色模式，支持随心切换。

- **⚡ High Performance / 高性能**  
  Optimized list rendering and GPU-accelerated animations ensure zero-lag experience.  
  优化的列表渲染与 GPU 加速动画，确保零卡顿的操作体验。

---

### 🛠 Tech Stack / 技术栈

- **Frontend**: Vue 3 (Composition API), Vite, Vanilla CSS
- **Desktop**: Electron
- **Email Engines**: ImapFlow, Nodemailer, Mailparser
- **Typography**: Inter (Google Fonts)

### 🚀 Getting Started / 快速开始

#### Prerequisites / 前置准备
- [Node.js](https://nodejs.org/) (v18+)
- [pnpm](https://pnpm.io/)

#### Installation / 安装
```sh
pnpm install
```

#### Development / 开发环境
```sh
# Run Vite dev server and Electron
pnpm dev
```

#### Production Build / 打包构建
```sh
# Build for Linux and Windows
pnpm electron:build
```
